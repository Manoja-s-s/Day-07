## â• 2. Understanding the `+` Operator  
How Java treats `+` based on operands:

- num + num = Addition  
- num + String = Concatenation  
- String + num = Concatenation  
- String + String = Concatenation  

### Examples:
```java
10 + 20       // 30 (Addition)
10 + "A"      // "10A" (Concatenation)
"A" + 10      // "A10" (Concatenation)
"A" + "B"     // "AB" (Concatenation)
```
